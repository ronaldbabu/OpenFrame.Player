# OpenFrame Player

OpenFrame Player is a lightweight media player built with C#, .NET, and WPF.

## Features

- Modern WPF UI
- MVVM architecture
- Video playback
- Playback controls
- Volume control
- Seek bar
- Subtitle support (planned)
- Playlist support (planned)

## Tech Stack

- .NET 10
- C#
- WPF
- MVVM
- FFmpeg (planned)
- xUnit

## Getting Started

```bash
git clone https://github.com/yourusername/OpenFramePlayer.git
cd OpenFramePlayer
dotnet build
dotnet run --project OpenFrame.Player.UI
```
